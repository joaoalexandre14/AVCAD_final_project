# Socio-Agrarian Landscape Dynamics in Alentejo (NUTS II)

An econometric and spatial analysis examining how demographic aging and formal educational deficits among farm managers influence land-use choices (temporary vs. permanent crop systems) across 299 civil parishes in the Alentejo region, Portugal.

## Project Overview
This repository contains the complete analytical pipeline for the project, mapping the demographic-educational vulnerability core and modeling agricultural specialization using Python (OLS Regression, ANOVA) and GIS (Choropleth Mapping via QGIS).

## Key Findings
- **The Aging Impact:** A 10 percentage-point increase in the parish agricultural aging index is associated with an estimated 9 percentage-point reduction in temporary crop prevalence.
- **The Education Barrier:** While aging shifts land use towards permanent crops, low formal education acts as a conservative structural counterweight, anchoring older cohorts to traditional annual cycles due to technological barriers.
